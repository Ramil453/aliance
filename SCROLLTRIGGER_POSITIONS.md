# Что означает "top" и процент в ScrollTrigger

## Формат позиции

```
start: 'triggerPosition viewportPosition'
```

## Первая часть: позиция на элементе-триггере

**`top`** - это позиция на элементе услуги (блоке, который указан в `trigger:`)

Варианты:
- `top` - верх элемента
- `center` - центр элемента
- `bottom` - низ элемента

**Пример:**
```typescript
trigger: firstServiceElement,  // ← это блок услуги
start: 'top ...'                // ← "top" означает ВЕРХ этого блока
```

## Вторая часть: позиция на экране (viewport)

**Процент** - это позиция на видимой области экрана (viewport)

### Как считаются проценты:

- `0%` или `top` - самый верх экрана
- `50%` или `center` - центр экрана
- `100%` или `bottom` - низ экрана
- `120%` - ниже низа экрана (на 20% за пределами видимой области)

### Визуально:

```
┌─────────────────┐ ← 0% (top) - верх экрана
│                 │
│                 │ ← 25% от верха
│                 │
│                 │ ← 50% (center) - центр экрана
│                 │
│                 │ ← 75% от верха
│                 │
│                 │ ← 100% (bottom) - низ экрана
└─────────────────┘
                  │ ← 120% - ниже экрана (не видно)
                  │
```

## Примеры

### Пример 1: `start: 'top bottom'`

```
│                 │
│                 │
│                 │
│                 │
│                 │
│                 │
│  ┌───────────┐  │ ← Низ экрана (100%)
│  │ УСЛУГА    │  │ ← Когда ВЕРХ блока здесь → start: 'top bottom'
│  └───────────┘  │
└─────────────────┘
```

```
│  │ УСЛУГА    │  │ ← Когда ВЕРХ блока здесь → start: 'top 80%'
│  │           │  │
│  └───────────┘  │
│                 │ ← Низ экрана (100%)
└─────────────────┘
```

### Пример 3: `start: 'top 120%'`

```
  └───────────┘  │
```

## В вашем коде

```typescript
start: 'top 120%'
```

Это означает:
- **`top`** - верх блока первой услуги
- **`120%`** - когда этот верх находится на 120% от верха экрана (то есть ниже видимой области)

**Результат:** Анимация начнется позже, когда пользователь проскроллит ниже, и верх первой услуги окажется за пределами видимой области.

## Таблица значений

| Значение | Позиция на экране |
|----------|-------------------|
| `top` или `0%` | Самый верх экрана |
| `25%` | Четверть от верха |
| `center` или `50%` | Центр экрана |
| `75%` | Три четверти от верха |
| `bottom` или `100%` | Низ экрана |
| `120%` | Ниже низа экрана (не видно) |
| `150%` | Еще ниже |

## Как выбрать значение

- **Чтобы анимация начиналась раньше:** используй меньший процент (`top`, `50%`, `80%`)
- **Чтобы анимация начиналась позже:** используй больший процент (`100%`, `120%`, `150%`)
