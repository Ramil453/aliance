# Где менять значения для страницы Services

## 1. Первая картинка - момент начала анимации

**Файл:** `app/services/page.tsx`  
**Строки:** 178-181

```typescript
ScrollTrigger.create({
  trigger: firstServiceElement,
  start: 'top bottom',    // ← ЗДЕСЬ: когда начинается анимация
  end: 'top center',     // ← ЗДЕСЬ: когда заканчивается анимация
  // ...
});
```

**Варианты для `start`:**
- `'top bottom'` - когда верх первой услуги достигает низа экрана (самое раннее)
- `'top 80%'` - когда верх достигает 80% высоты экрана
- `'top center'` - когда верх достигает центра экрана
- `'top 20%'` - когда верх достигает 20% высоты экрана (позже)

**Варианты для `end`:**
- `'top center'` - когда верх достигает центра
- `'bottom center'` - когда низ достигает центра
- `'top 50%'` - когда верх на 50% высоты

## 3. Скорость анимации

**Файл:** `app/services/page.tsx`  
**Строки:** 171-176 (первая картинка) и 208-213 (остальные)

```typescript
const firstAnimation = gsap.timeline()
  .to(topImage, {
    clipPath: 'inset(0% 0% 0% 0%)',
    duration: 1,        // ← ЗДЕСЬ: длительность анимации (в секундах)
    ease: 'none',       // ← ЗДЕСЬ: тип анимации
  });
```

**Варианты для `duration`:**
- `0.5` - быстрее
- `1` - нормально
- `1.5` - медленнее
- `2` - очень медленно

**Варианты для `ease`:**
- `'none'` - линейная (равномерная)
- `'power2.inOut'` - плавная
- `'power2.out'` - ускорение в конце
