# Как читать маркеры ScrollTrigger

## Что такое маркеры?

Маркеры - это визуальные линии на экране, которые показывают **точные моменты**, когда анимация начинается и заканчивается.

## Типы маркеров

### 1. Зеленый "start" маркер
- **Что означает:** Здесь начинается анимация
- **Когда срабатывает:** Когда элемент достигает этой позиции на экране

### 2. Красный "end" маркер
- **Что означает:** Здесь заканчивается анимация
- **Когда срабатывает:** Когда элемент достигает этой позиции на экране

### 3. "scroller-start" и "scroller-end"
- Показывают начало и конец области скролла для анимации
- Появляются когда используется `scrub: true`

## Как читать маркеры на вашем скриншоте

### Маркеры вверху справа (около текстового блока):

```
┌─────────────────┐
│                 │
│  ┌───────────┐  │ ← Зеленый "start" - здесь начинается анимация
│  │ ТЕКСТ     │  │
│  │           │  │
│  │           │  │
│  └───────────┘  │ ← Красный "end" - здесь заканчивается анимация
│                 │
└─────────────────┘
```

**Что это значит:**
- Когда верх текстового блока достигает зеленой линии → анимация начинается
- Когда низ текстового блока достигает красной линии → анимация заканчивается

### Маркер внизу справа (около первой услуги):

```
│                 │
│  ┌───────────┐  │
│  │ УСЛУГА 1  │  │
│  └───────────┘  │
│                 │
└─────────────────┘ ← Зеленый "start" - здесь начинается анимация первой картинки
```

**Что это значит:**
- Когда верх первой услуги достигает этой зеленой линии → первая картинка начинает выезжать

## Как использовать маркеры для настройки

### Проблема: маркер слишком высоко
**Решение:** Увеличь процент в `start`
```typescript
start: 'top 80%'  →  start: 'top 100%'  →  start: 'top 120%'
```

### Проблема: маркер слишком низко
**Решение:** Уменьши процент в `start`
```typescript
start: 'top 120%'  →  start: 'top 100%'  →  start: 'top 80%'
```

### Проблема: анимация слишком быстрая
**Решение:** Увеличь расстояние между `start` и `end`
```typescript
start: 'top bottom',
end: 'top center',     // Маленькое расстояние = быстро

start: 'top bottom',
end: 'bottom top',     // Большое расстояние = медленно
```

### Проблема: анимация слишком медленная
**Решение:** Уменьши расстояние между `start` и `end`
```typescript
start: 'top bottom',
end: 'bottom top',     // Большое расстояние = медленно

start: 'top bottom',
end: 'top center',     // Маленькое расстояние = быстро
```

## Практический пример

На вашем скриншоте видно:
- Зеленый "start" внизу справа около текста услуги

Это означает, что в коде у вас:
```typescript
start: 'top 120%'  // или похожее значение
```

**Если маркер слишком низко:**
- Измени на `start: 'top 100%'` (низ экрана)
- Или `start: 'top 80%'` (выше)

**Если маркер слишком высоко:**
- Измени на `start: 'top 150%'` (еще ниже)
- Или `start: 'top 200%'` (намного ниже)

## Визуальная схема

```
┌─────────────────┐ ← 0% (верх экрана)
│                 │
│  ┌───────────┐  │ ← 50% (центр экрана)
│  │           │  │
│  │           │  │
│  └───────────┘  │
│                 │
│                 │ ← 100% (низ экрана)
└─────────────────┘
                  │ ← 120% (ниже экрана) ← Зеленый маркер здесь
                  │
  ┌───────────┐   │
  │ УСЛУГА    │   │
  └───────────┘   │
```

## Главное правило

**Маркер показывает ТОЧНУЮ позицию, где срабатывает анимация.**

- Хочешь, чтобы анимация начиналась раньше → перемести маркер выше (уменьши процент)
- Хочешь, чтобы анимация начиналась позже → перемести маркер ниже (увеличь процент)
